# tiny_machine_learning
​        基于西瓜书写的一些常见的机器学习算法的基础实现，并与sklearn中集成好的相关算法做对比，还有一些结果可视化的展示。

包含以下算法：

1. 线性回归
2. 逻辑回归
3. 线性判别分析（LDA）
4. 决策分类树：ID3（基于信息增益）、C4.5（基于信息增益率）、CART（基于基尼指数）
5. 决策回归树：基于均方误差
6. 神经网络BP算法
7. 朴素贝叶斯
8. AdaBoost
9. RandomForest
10. GradientBoosting
11. K-means & K-means++
12. PCA降维
